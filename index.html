Home K3s cluster with internal DNS & internal CA

Repo is WIP and not on github yet

Automated bootstrapping of K3s and base cluster services for my Kubernetes homelab:

  • Calls existing ansible role for K3s
  • Custom ansible roles to automate provisioning of base cluster services
    • MetalLB - L4 load balancing across multiple internal hosts
    • Traefik - Reverse proxy for each service, i.e. service.int.example.com
    • CoreDNS - Create A record for each new service
    • StepCA - Self-hosted certificate authority
